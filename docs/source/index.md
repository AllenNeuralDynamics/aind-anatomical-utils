# AIND Anatomical Utils

Tools to work with anatomically-aware files

## Installation

```bash
pip install aind-anatomical-utils
```

For development:
```bash
git clone https://github.com/AllenNeuralDynamics/aind-anatomical-utils.git
cd aind-anatomical-utils
uv sync
```

## Quick Start

```python
import aind_anatomical_utils

# Your code here
```
